HTML Cheat Sheet

Tags Names Description Reference Link
<li> List element The <li> item element creates list items inside ordered lists<ol> and <ul>. Codecademy
<video> Video element The <video> element embeds a media player for video playback. The <src> attribute will contain the URL to the video, including other attributes. Codecademy
<img> The img element HTML images <img> element embeds images into documents. The <src> attribute contains the image URL and is mandatory. Codecademy
<div> The div element The <div> element is used as a container that divides HTML documents into sections and is short for "division" Codecademy
<h1>-<h6> Heading Elements HTML can use six different levels of heading elements. The heading elements are ordered from the highest level <h1> to the lowest level <h6>. Codecademy
<p> The paragraph element The <p> paragraph element contains and displays a block of text. Codecademy
<span> The span element The <span> element is an inline container for text and can be sued for group text for styling purposes. Codecademy
<strong> The strong element The strong<strong> element highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold default. Codecademy
<a> The Anchor element The <a> element is used to create hyperlinks in an HTML document. The hyperlinks can point to other webpages. Codecademy
<head> The head element The <head> element contains general information about an HTML page that isn't displayed on the page itself. Codecademy
